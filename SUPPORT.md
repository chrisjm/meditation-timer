# Support

If you are using this meditation timer and run into an issue:

- If you have access to this repository, open an issue with:
  - Steps to reproduce
  - Expected behavior
  - Actual behavior
  - Browser/device information
- Otherwise, use the support contact listed in the app UI (email or contact form).

## Maintenance expectations

- Best-effort support from the maintainer.
- Breaking changes will be released as a new major version and documented in `CHANGELOG.md`.
- Notable bug fixes and feature additions will be recorded in `CHANGELOG.md`.
